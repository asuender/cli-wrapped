# cli-wrapped

A small TUI that beautifully summarizes your terminal usage!

## Install

```bash
npm install --global cli-wrapped
```

## Development Setup

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Build the project and link it globally:
   ```bash
   pnpm run build
   pnpm link --global
   ```

Now you can run `cli-wrapped` from anywhere.

For development with auto-rebuild on changes:

```bash
pnpm run dev
```

## Usage

```bash
cli-wrapped --help
```
